
# Generador de Nombres RPG

Repositorio:
https://github.com/dbausada-art/RPG_Character-Name-Generator

## Sobre el proyecto

Este es uno de los proyectos que fui desarrollando mientras aprendo Python.

La idea surgió porque siempre me gustaron los juegos de rol y me pareció una buena excusa para practicar programación creando algo útil y entretenido. El programa permite generar nombres para personajes de distintas razas, elegir si serán masculinos o femeninos y definir la cantidad de sílabas que tendrán los nombres generados.

Además, los resultados pueden guardarse en un archivo de texto para consultarlos más adelante o utilizarlos en partidas, historias o cualquier otro proyecto.

## Qué puede hacer

* Elegir una raza disponible.
* Generar nombres masculinos o femeninos.
* Definir una cantidad mínima y máxima de sílabas.
* Crear varios nombres en una sola ejecución.
* Evitar nombres repetidos.
* Guardar los nombres generados en un archivo `.txt`.

## Requisitos

* Python 3.10 o superior
